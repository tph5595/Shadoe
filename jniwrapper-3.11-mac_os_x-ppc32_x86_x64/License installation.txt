Installing License File

--------------------------
The JNIWrapper library requires valid licence file: jniwrap.lic.

If you had purchased the product, the licence was sent to you by email.

If you are evaluating our product, you need an evaluation licence. To get the evaluation 
licence please complete the following steps:
	- click the "Evaluate now" button on the product page (http://www.teamdev.com/jniwrapper/); 
	- fill your name and email in the evaluation form;
	- press the "Send me evaluation key" button.

The link to evaluation licence download will be emailed to you.

Licence file has to be installed in the same directory as the native code library or in 
the META-INF subfolder of any jar library from the application's classpath.

Do NOT rename the licence file or it will not be recognized.
