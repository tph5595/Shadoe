MacPackDemo is an application, which  demonstrates commonly used features of the
MacPack library. In order to run MacPackDemo you should have the following files
that are not distributed with MacPack:

1. Required files (distributed with JNIWrapper):

   - JNIWrapper library (jniwrap-3.9.jar);
   - JNIWrapper native library (libjniwrap.jnilib); 
   - JNIWrapper license file (jniwrap.lic).
   
All these files should be located in the "demo" subfolder.
