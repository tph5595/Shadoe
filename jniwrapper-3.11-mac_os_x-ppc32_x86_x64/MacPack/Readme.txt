2. WHAT IS MACPACK

MacPack for JNIWrapper is a first free addition to  JNIWrapper  that  opens  the
door to MacOS X for your Java software.

MacPack is a library that you can  use  along  with  JNIWrapper  to  extend  its
abilities.

Currently in MacPack:
I. Dialogs
  - Wrapper for native Open/Save file dialogs;
  - Wrapper for native Select Folder dialogs;
II. Window
  - Window Decoration: on-top, transparent windows, tool windows;
III. IO
  - File  System Watcher:  functionality that  allows listening  to file  system
  events;
IV. ObjectiveC
  - Support for Cocoa classes.

etc.

3. GETTING STARTED

The  MacPack  package  downloadable  from  JNIWrappper  Download  page  contains
following:
 - MacPack Library (lib/macpack.jar)
 - MacPack License file (License.txt)
 - This Readme file (Readme.txt)
 - MacPackDemo application (macpackdemo.jar library, bin/MacPackDemo.sh)

   Installing Library JAR File
   ---------------------------
MacPack JAR file should be on the program's class path.
The library file can also be placed on the boot class path or in the extension
directory of Java runtime, but this is not required.
